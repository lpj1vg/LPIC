# REGEX

## Markdown treats these characters as ordinary text if there is backslash escape character in front of them:


| \\\      | backslash itself. |
| \\`      | backtick.         |
| \\*      | asterisk.         |
| \\_      | underscore.       |
| \\{ \\}  | curly braces.     |
| \\[ \\]  | square brackets.  |
| \\( \\)  | parentheses       |
| \\#      | hash mark         |
